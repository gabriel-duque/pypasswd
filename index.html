Password Generator¶

generator.py:

This module implements the PasswdGenerator class which does the actual password generation and a main function which is the entrypoint to a simple program to generate a password.

__call__(length: int = 64, no_strict: bool = False) → str¶

Actually generate a password. It is important to check at least one character from every character set was included in the password.

This function raises a TypeError if length is not an integer of strict is not a boolean.

A ValueError is raised if the asked length is smaller of equal to 0 or if strict is True and the wanted length is smaller than the length of the character set list.

Parameters

  • length (int) – length of the password to generate

  • no_strict (bool) – do not enforce strict generation

Returns

generated password

Return type

str

pypasswd.generator.main() → None¶

This function is to be called as a script using the pypasswd command.

main will parse arguments, create a PasswdGenerator object then call it as many times as required to get passwords.
